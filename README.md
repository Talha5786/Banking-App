## Project Description
This project implements a simple banking application in Python.
It includes unit tests and integration tests written using pytest.
GitHub Actions is used to automatically run tests on every push.

## How to Run Tests
Install dependencies:
pip install -r requirements.txt

Run all tests:
pytest

Run tests with details:
pytest -v

Generate HTML report:
pytest --html=report.html -v

## GitHub Actions
GitHub Actions automatically runs all unit and integration tests
on every push or pull request to ensure code quality.
